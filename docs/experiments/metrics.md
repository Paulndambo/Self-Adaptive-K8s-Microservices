# Metrics

The evaluation layer computes metrics for performance, resource usage, adaptation behavior, stability, and cost.

## Performance

- Average latency
- Median latency
- p95 latency
- SLA violation rate

## Resource Usage

- Pod-seconds
- Average replicas
- Peak replicas

## Adaptation

- Scaling event count
- Scale-up count
- Scale-down count
- Average adaptation latency
- Maximum adaptation latency

## Stability and Cost

- Oscillation count
- Replica variance
- Stability score
- Replica cost
- Waste cost
